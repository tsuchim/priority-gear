## Safety boundary

PriorityGear System Mode is privileged software. It installs a LocalSystem Windows Service and changes process priority, which can affect system responsiveness or stability.

This software is provided AS IS. Use it at your own risk.

- shared-host `svchost.exe` mutation is not enabled by default
- executable-only `svchost.exe` rules are not treated as safe
- service-name rules require administrator approval
- protected or denied targets are reported explicitly
- PriorityGear does not bypass Windows protected-process boundaries
