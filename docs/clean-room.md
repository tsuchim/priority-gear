# Clean-Room Policy

PriorityGear is an original implementation.

## Allowed

- Behavior-level compatibility based on public descriptions.
- General Windows API behavior from official Microsoft documentation.
- Original code, UI, documentation, icons, and resources.

## Forbidden

- Reverse engineering AutoGear.
- Decompiling AutoGear.
- Copying AutoGear code.
- Copying AutoGear binary resources, icons, UI assets, or text.
- Copying AutoGear UI strings beyond generic terms such as "priority", "active", and "normal".
- Importing old freeware assets.

When behavior is unclear, use official Microsoft documentation or define explicit original behavior.
